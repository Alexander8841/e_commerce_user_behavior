# E-commerce User Behavior & Cohort Analysis

## Project Goal
Analyze customer behavior in an e-commerce platform to understand retention, purchasing patterns, and key drivers of business performance.

---

## Data Description
The dataset includes:
- orders
- customers
- order items

---

## Objectives
- Analyze purchasing frequency
- Perform cohort analysis
- Calculate retention metrics
- Conduct RFM segmentation
- Identify delivery issues and their impact

---

## Tools & Technologies
- Python (pandas, numpy)
- Cohort analysis
- RFM segmentation
- Data aggregation
- Visualization (seaborn, matplotlib)

---

## What Was Done
- Cleaned and joined multiple datasets
- Calculated purchase frequency per customer
- Built cohort tables and retention curves
- Performed RFM segmentation:
  - Recency
  - Frequency
  - Monetary
- Analyzed delivery failures and delays

---

## Key Results
- Identified retention patterns across cohorts
- Segmented customers by value and behavior
- Detected issues affecting order completion
- Highlighted potential areas for business growth

---

## Skills Demonstrated
- Cohort analysis and retention calculation
- Customer segmentation (RFM)
- Business-oriented data analysis
- Working with transactional data
- Identifying product and operational issues
